# 五子棋游戏

一个简单而优雅的五子棋游戏，使用纯 HTML、CSS 和 JavaScript 实现，无需任何依赖，可直接在浏览器中运行。

## 项目简介

这是一个经典的五子棋（Gomoku）游戏实现，支持双人对战。游戏界面美观，操作简单，适合所有年龄段的玩家。

## 功能特性

- 🎮 经典五子棋玩法
- 🎨 现代化的用户界面，带有流畅的动画效果
- 📱 响应式设计，完美支持移动端和桌面端
- ⚡ 纯前端实现，无需后端服务器
- 🎯 自动检测胜负
- 📊 游戏统计功能，记录胜负次数
- 🔄 一键重新开始
- ❓ 内置游戏说明和帮助

## 使用方法

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件
3. 开始游戏！黑棋先行，点击棋盘上的交叉点放置棋子
4. 率先在横、竖、斜任意方向连成五子的玩家获胜

## 游戏规则

- 黑棋先行
- 双方轮流在棋盘上放置棋子
- 率先在横、竖、斜任意方向连成五子的玩家获胜
- 点击"重新开始"按钮可以随时开始新一局
- 游戏会自动记录并显示双方的获胜次数
- 点击右下角的"?"按钮可以查看详细的游戏说明

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
