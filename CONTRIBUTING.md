# 贡献指南

感谢您对五子棋游戏项目的关注！我们欢迎所有形式的贡献。

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请通过以下方式提交：

1. 在 GitHub 上创建一个新的 Issue
2. 详细描述问题或建议
3. 如果可能，请提供复现步骤

### 提交代码

1. Fork 本项目
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

### 代码规范

- 使用清晰的变量和函数命名
- 添加必要的注释
- 保持代码简洁易读
- 遵循现有的代码风格

### 提交信息规范

提交信息应该清晰描述所做的更改：

- `feat: 添加新功能`
- `fix: 修复bug`
- `docs: 更新文档`
- `style: 代码格式调整`
- `refactor: 代码重构`
- `test: 添加测试`

## 开发环境设置

1. 克隆仓库：`git clone https://github.com/yourusername/wuziqi.git`
2. 进入目录：`cd wuziqi`
3. 直接在浏览器中打开 `index.html` 即可开始开发

## 代码风格

- 使用2个空格缩进
- 使用单引号
- 函数名使用驼峰命名
- 添加必要的注释
- 保持代码简洁易读

## 需要帮助？

如果您有任何问题，请随时创建 Issue 或联系维护者。

再次感谢您的贡献！

## 代码审查

提交的代码会经过审查，请耐心等待。

## 测试

提交前请确保代码可以正常运行。

## 提交规范

请使用清晰的提交信息。

## 分支管理

建议使用功能分支进行开发。

## 合并请求

提交PR前请确保代码质量。
